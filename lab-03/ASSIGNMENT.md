## Assignment

The original assignment was this:

1. Install the IDE [STM32CubeIDE] and create a new project. This same project will be used to developing code from module 3 to 8. 
Name the project with format of "[REDACTED]_Firstname_Lastname". Check the "homework assignment submission instructions.pdf" 
for detailed guidelines for code submission. (10%)

2. Implement the printf() redirection and print your name on Tera term every 1s,
with help of the HAL delay function. (40%)

3. Toggle the onboard LED every time the program prints 
(You can find how to do this from the reading). (40%)

4. Add git repository to your project and make commits to record your progress. (10%)

5. Demonstrate your project to TA on the visit day. We would like to check if you have a dedicated workspace 
to save your project folder, if there is a local git repository, if the git is correctly setup, if you can 
manage your IDE, if the code runs correctly and if you understand how it works.

6. Make sure your submission is completed and can be build by the instructor. The instructor will download your project submission and check by himself. if the code does not build or work, you will lose half of the project/demo points, which is 30% of that module.
