# nucleo-g071rb - Board Support Package

Board support package for the ST NUCLEO-G071RB, providing register definitions, GPIO/UART drivers, startup code, and a linker script for bare-metal Zig on the STM32G071RB (Cortex-M0+).

## Register Definitions

The files under `src/chip/types/peripherals/` were generated from the STM32G071 SVD file (`STM32G071.svd`)
using [regz](https://github.com/ziglang/zig/tree/master/tools/regz). The SVD file describes the chip-specific peripheral 
register layout (GPIO, USART, RCC, timers, etc.) and regz converts it into type-safe Zig `packed struct` definitions backed by `mmio.Mmio`.

The SVD file is sourced from [STMicroelectronics' CMSIS pack](https://github.com/cmsis-svd/cmsis-svd/blob/main/data/STMicro/STM32G071.svd) and is licensed under Apache-2.0.

To regenerate (if the SVD or regz changes):

```sh
regz STM32G071.svd --output src/chip/
```

### Manual Additions

The SVD only covers chip-specific peripherals. The following ARM core peripherals are not
present in the SVD and were added manually, following the same `mmio.Mmio` pattern used by the generated code:

- **SysTick** (`src/chip/types/peripherals/SysTick.zig`) - The ARM Cortex-M System Timer is defined at a 
fixed address (`0xE000E010`) across all Cortex-M0+ cores. Its register layout (CSR, RVR, CVR, CALIB) comes
from the [ARM Cortex-M0+ Technical Reference Manual](https://developer.arm.com/documentation/dui0662/b/Cortex-M0--Peripherals/System-timer--SysTick).

The SysTick peripheral instance is declared in `src/chip/STM32G071.zig` alongside the regz-generated peripheral instances.

## Structure

```
build.zig              Exports stm32_target and the "board" module
build.zig.zon          Package metadata
STM32G071.svd          Source SVD (from ST)
stm32g071rb.ld         Linker script (128K flash, 36K RAM)
src/
  board.zig            Public API - re-exports Hardware, gpio, uart, startup, chip
  hardware.zig         Board-level init: clocks, GPIO mux, UART, SysTick
  [...]                Various drivers, and helpers that use definitions from chip/types
  startup.zig          _start (reset handler), vector table, .data/.bss init
  mmio.zig             Generic MMIO register access (read/write/modify)
  chip/
    STM32G071.zig      Peripheral base addresses, interrupt table, vector table struct
    types.zig          Re-exports types/peripherals.zig
    types/
      peripherals.zig  Index of all peripheral type modules
      peripherals/     One file per peripheral type (generated by regz + manual additions)
```
